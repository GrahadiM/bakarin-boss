## Installation
### Clone Repo

```bash
git clone https://github.com/GrahadiM/bakarin-boss.git
```
### Go to folder

```bash
cd bakarin-boss
```
### Install and build App with npm

```bash
npm install
```
and
```bash
npm run dev
```
### Run App

```bash
npm run start
```
